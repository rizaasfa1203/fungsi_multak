Bentuk Umum Fungsi Eksponensial

Fungsi eksponensial memiliki bentuk umum:

y = a · bx atau f(x) = a · bx

Syarat penting:

  • a adalah konstanta (a ≠ 0)
  • b adalah bilangan pokok/basis, dengan syarat b > 0 dan b ≠ 1
  • x adalah variabel bebas (eksponen/pangkat)

Langkah-Langkah Menggambar Grafik

Untuk menggambar grafik fungsi eksponensial, ikuti langkah mudah ini:

  1. Buat Tabel Titik Bantu: Pilih beberapa nilai x (misalnya -2, -1, 0, 1, 2) dan hitung nilai y yang bersesuaian.
  2. Gambarkan Titik: Letakkan titik-titik (x, y) yang didapat ke dalam sistem koordinat Kartesius.
  3. Hubungkan Titik: Tarik garis kurva mulus (tidak patah-patah) yang menghubungkan titik-titik tersebut.
  4. Perhatikan Asimtot: Pastikan kurva mendekati sumbu X (atau garis asimtot lainnya) namun tidak pernah menyentuhnya.

Sifat-Sifat Grafik Fungsi Eksponensial y = bx

  • Selalu memotong sumbu Y di titik (0, 1), karena b0 = 1.
  • Selalu berada di atas sumbu X (karena nilai bx selalu positif).
  • Mempunyai asimtot datar yaitu sumbu X (garis y = 0).
  • Monoton Naik: Jika b > 1, grafik naik dari kiri ke kanan.
  • Monoton Turun: Jika 0 < b < 1, grafik turun dari kiri ke kanan.
